# 安卓定时震动提醒 APP
<img src="demo.jpg" height="200px" />

## 应用简介

本应用是一款简单易用的安卓定时震动提醒工具。用户可自定义震动间隔、每次震动次数和循环次数，适用于久坐提醒、定时锻炼、定时任务等场景。支持后台运行，参数设置自动保存，下次打开自动恢复。

---

## 主要功能

- **定时震动提醒**  
  按用户设定的时间间隔自动震动，支持秒/分钟单位。
- **自定义震动次数**  
  每次震动可设置多次，满足不同提醒需求。
- **循环次数设置**  
  可设置震动循环次数，支持无限循环。
- **一键启动/停止**  
  主界面按钮随时开启或关闭定时震动服务。
- **参数自动保存**  
  设置参数自动持久化，下次打开自动恢复。
- **后台运行**  
  服务开启后即使APP退到后台也能持续震动提醒。
- **权限管理**  
  启动时自动申请震动权限。

---

## 使用方法

1. 打开APP，设置震动间隔、次数、循环次数和提示内容。
2. 点击“应用设置”保存参数。
3. 点击“开始”按钮启动定时震动服务。
4. 需要停止时点击“停止”按钮即可。

---

## 技术说明

- 原生安卓开发，Java语言
- 使用系统 `Vibrator` API 实现震动
- 定时任务采用服务方式，支持后台运行
- 参数持久化采用 SharedPreferences
- 最低支持 Android 5.0（API 21）

---

## 适用场景

- 定时锻炼
- 定时任务/学习提醒
- 其他需要定时震动提示的场合
