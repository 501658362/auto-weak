# 安卓定时震动 APP 需求说明

## 简介

本安卓 APP 旨在实现**定时震动**功能。用户可以自定义震动间隔、每次震动的次数，并可通过开关按钮控制是否启用定时震动服务。

---

## 功能需求

1. **定时震动**
    - 用户可设置每隔多久震动一次（支持秒或分钟间隔）。
2. **设置每次震动的次数**
    - 用户可以设置震动时连续震动几次（例如：每次震动3下）。
3. **开关控制**
    - 提供开关按钮，用户可以随时开启或关闭定时震动服务。
4. **简洁的界面**
    - 主页面包含设置项与开关，操作直观简洁。
5. **后台运行**
    - 开启服务后，即使APP退到后台也能按设置定时震动。
6. **权限管理**
    - 启动APP时需申请震动权限。

---

## 界面设计

- 震动间隔设置（输入框/选项选择器）
- 每次震动次数设置（输入框/选择器）
- 启动/关闭的开关按钮（显示当前状态）
- （可选）简洁的设置页或仅主界面展示全部操作项

---

## 技术建议

- 针对安卓平台，语言可选Java
- 使用系统震动 API（`Vibrator`）
- 定时任务可通过 `AlarmManager`、`WorkManager` 或 `Handler` 实现
- Android 最低支持API可设置为21及以上
- 推荐使用原生View或Jetpack Compose实现UI
- 在服务关闭或APP退出时应正常取消定时震动任务

---

## 其他说明

- 设置参数调整后应立即生效
- 用户关闭定时震动后，后台定时任务应立即停止
- APP逻辑应尽量简洁明了，便于维护
- 建议主界面为 MainActivity
